Links to other pages

Links are easy - just put them inside square brackets [~like so] and they become full links [like so] after editing, and when presented for the browser.
If the page does not exist at the outset, Cozmos will give you a chance to add a page. It does this by shwoing a standard template instead of showing a "page not found" error in the browser.

Spaces in wiki style links will be turned into underscores ('_') automatically in the resulting file name. Click on the left (to the left) to see the link transformed.

Hierachical directories are easy too. Something like [~test/an example] becomes [test/an example] after editing. This example suggesting there is a file calles 'an_example.html' inside a folder called 'test'. You can group you content as you like.
Giving your links descriptions is easy too. Put a bar ('|') between the description and the pseudo file name. Thus [~hello how are you|greeting] and [~howdie|greeting] become [hello how are you|greeting] and [howdie|greeting].

Suport for other HTML features

Tables, indented sections, preformated sections headings and embedded images are all possible. Your HTML editor probably has the menu options to effect them built in. For images, wrap around text looks nice. Images will be copied to the Cozmos server from any location (local file system or another website).
